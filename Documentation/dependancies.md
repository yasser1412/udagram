# App Dependencies

## Dependencies

```text
    - Node v16 (LTS) or more recent.

    - npm 5.5.1 (LTS) or more recent.

    - AWS CLI v2

    - AWS EB CLI

    - AWS RDS database running Postgres.

    - AWS S3 bucket for Frontend.

    - AWS Elastic Beanstalk for Backend.
```
